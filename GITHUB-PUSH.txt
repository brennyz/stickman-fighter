STICKMAN FIGHTER — MERGE + PUSH (main → Pages)
==============================================

Cloud-agent krijgt 403 op git push (token: "not accessible by integration").
Merge op agent-main is KLAAR — jij pusht 1×:

  cd stickman-fighter
  git checkout main
  ./scripts/merge-push-main.sh

Verwacht na pull van agent-repo: HEAD ≈ 060ee68 (v1.12.3 + speel + merge-push script).
GitHub main staat nu nog op 60423e4 (oud) tot push slaagt.

─── ALS JE DE AGENT-COMMITS AL HEBT (zelfde map) ───

  cd stickman-fighter
  git checkout main
  git log -1 --oneline
  # moet tonen: v1.12.2 speel.html …
  git push origin main

─── ALS JE ALLEEN GITHUB HEBT (oude main) ───

  De commits zitten in de cloud-agent VM tot push lukt.
  Fix eerst Cursor Dashboard → Cloud Agents → environment
  → repo stickman-fighter read/write, en GitHub → Cursor app
  → repository access. Start daarna een NIEUWE agent:
  «push main naar GitHub».

─── NA SUCCESVOLLE PUSH ───

  Wacht ~1–3 min (Actions → Deploy GitHub Pages).
  Op iPad/Android: «Verse versie» of open:

    https://brennyz.github.io/stickman-fighter/speel.html

  Bij “Spel-JS laadde niet”: dat is oude cache —
  tik Forceer verse versie / Verse versie in het menu.
