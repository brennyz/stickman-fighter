STICKMAN FIGHTER — IPAD (geen terminal, geen “command”)
======================================================

Alles wat je op iPad nodig hebt = Safari + tikken. Terminal/command is alleen
op een Mac/PC (of in Cursor op je computer), niet op de iPad zelf.


─── SPELEN (vaste bookmark, URL verandert niet) ───

1) Open Safari.

2) Typ of plak in de adresbalk (één van deze — beide blijven hetzelfde):

   TUNNEL (als thuis iemand de tunnel aan heeft staan):
   https://stickfighter-ipad-b75e.loca.lt/ipad.html

   GITHUB PAGES (werkt overal, geen tunnel nodig — na push naar GitHub):
   https://brennyz.github.io/stickman-fighter/ipad.html

3) Eerste keer bij loca.lt: soms gele knop “Click to Continue” — tik die.

4) Onderaan in het menu moet een versie staan (bv. v1.11.5 · SW v60).
   Zie je oude versie? Tik in het menu op «Verse versie» (wist cache).

5) App-icoon op beginscherm (optioneel):
   Safari → Delen (vierkant met pijl) → «Zet op beginscherm».
   Open daarna altijd via dat icoon — zelfde link als in stap 2.


─── MISSIES / SPELEN ───

• Menu → grote knoppen (Avontuur, Training, 2 spelers, …).
• Missies → claim XP als er een gouden rand om «Missies» staat.
• 2 spelers: iPad liggend, P1 links, P2 rechts.


─── TUNNEL “OFFLINE” OP IPAD? ───

De tunnel (loca.lt) draait op een computer thuis — niet op je iPad.
Is de link dood (503 / laadt niet)? Gebruik de GitHub Pages-link hierboven
of vraag iemand thuis de tunnel weer aan te zetten (Mac: zie onder).


─── WAT JIJ NIET OP IPAD DOET ───

• geen «git push»
• geen «./start-local.sh»
• geen Cursor-terminal

Dat doe je op Mac/PC, of laat een Cloud Agent / Cursor op je computer het doen.


─── THUIS (Mac) — tunnel aan, 3 stappen zonder command-kennis ───

1) Cursor of Terminal openen op de Mac (niet op iPad).
2) Map stickman-fighter openen.
3) Agent of helper laten draaien: «start tunnel» / start-local.sh --tunnel

Of: push naar GitHub → alleen Pages-link gebruiken op iPad (stap 2).


─── UPDATES VAN DE GAME ───

• Pages: na push op GitHub, even wachten (1–3 min), dan «Verse versie» op iPad.
• Tunnel: zelfde bookmark; thuis moet de nieuwste bestanden + tunnel draaien.

Vragen? Houd één bookmark vast (loca.lt óf github.io) — wissel niet steeds URL.
